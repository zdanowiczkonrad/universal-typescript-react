### Prerequisites

Have a Yarn installed. Then:

```
yarn global add typescript tslint
```

### Usage

```
yarn install
yarn start
open http://localhost:3000
```

### Building

A basic production script is included that builds your app to a `dist` folder

```
yarn prebuild
yarn build
yarn postbuild
```
